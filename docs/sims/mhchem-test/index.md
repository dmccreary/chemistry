## Inline Formulas

Water is $\ce{H2O}$ and table salt is $\ce{NaCl}$.

Sulfuric acid is $\ce{H2SO4}$ and hydrochloric acid is $\ce{HCl}$.

The sulfate ion is $\ce{SO4^{2-}}$ and the ammonium ion is $\ce{NH4+}$.

## Simple Reactions

$$\ce{2H2 + O2 -> 2H2O}$$

$$\ce{NaOH + HCl -> NaCl + H2O}$$

$$\ce{CaCO3 -> CaO + CO2}$$

## Equilibrium Reactions

$$\ce{N2 + 3H2 <=> 2NH3}$$

$$\ce{CH3COOH <=> CH3COO- + H+}$$

$$\ce{NH3 + H2O <=> NH4+ + OH-}$$

## State Symbols

$$\ce{NaCl (aq) + AgNO3 (aq) -> AgCl (s) v + NaNO3 (aq)}$$

$$\ce{CaCO3 (s) -> CaO (s) + CO2 (g) ^}$$

$$\ce{Zn (s) + 2HCl (aq) -> ZnCl2 (aq) + H2 (g) ^}$$
